# 📞 Call Center Trend Analysis Dashboard

## 📌 Project Overview

The Call Center Trend Analysis Dashboard is an interactive Power BI dashboard developed to monitor call center performance, agent efficiency, customer satisfaction, and case resolution metrics. The dashboard provides actionable insights that help management improve operational performance and customer experience.

---

## 🎯 Objective

- Monitor overall call center performance.
- Track answered and abandoned calls.
- Analyze resolved and unresolved cases.
- Measure customer satisfaction levels.
- Monitor average speed of answer.
- Support management with interactive business insights.

---

## 🛠️ Tools & Technologies

- Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

---

## 📂 Dataset

The dataset contains call center operational data, including:

- Call ID
- Agent Name
- Call Date
- Call Topic
- Call Status
- Resolved Status
- Customer Satisfaction Rating
- Speed of Answer (Seconds)
- Call Duration

---

## 📊 Dashboard Features

- 📞 Total Calls KPI
- 👨‍💼 Total Agents KPI
- ✅ Calls Answered
- ❌ Calls Abandoned
- ✔️ Resolved Cases
- ⚠️ Unresolved Cases
- ⏱️ Average Speed of Answer
- ⭐ Average Customer Satisfaction Rating
- 📈 Calls Trend by Day
- 😊 Customer Satisfaction Analysis
- 📅 Monthly Resolved Cases
- 🎛️ Interactive Filters
  - Day
  - Topic
  - Month

---

## 📈 Key Insights

- Total Calls Handled: **5,000**
- Calls Answered: **4,054 (81.08%)**
- Calls Abandoned: **946 (18.92%)**
- Resolved Cases: **3,646 (72.92%)**
- Unresolved Cases: **1,354 (27.08%)**
- Average Speed of Answer: **67.52 Seconds**
- Average Customer Satisfaction Rating: **3.40 / 5**
- Daily call trends help identify peak and low-volume days.
- Monthly analysis provides insights into resolved and unresolved cases.
- Customer satisfaction analysis highlights service quality and improvement areas.
